# RadioGestor 📡

**RadioGestor** é um sistema institucional para a gestão da infraestrutura de rádio comunicação, com foco em controle, visualização e manutenção de sítios com antenas de comunicação.

## 🚀 Objetivo

Oferecer uma ferramenta escalável e moderna para mapear, consultar e manter informações sobre:
- Localização georreferenciada dos sítios com antena
- Tipo de antena (Tier III, analógica, PRF, etc.)
- Municípios abrangidos
- Responsável contratual pela manutenção
- Proprietário do terreno e seus contatos
- Custos envolvidos com cada sítio (aluguel, corte de mato, etc.)

## 🔧 Tecnologias utilizadas

- Java 17+
- Gradle
- IntelliJ IDEA
- Git & GitHub

## 📌 Visão futura

- Interface gráfica amigável (JavaFX ou web)
- Integração com mapas interativos
- Painel de Business Intelligence (BI)
- Sistema de alertas e auditoria
